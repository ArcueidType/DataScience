### practice Four-4:

给出的代码应该是个选择排序，而非希尔排序，时间复杂度为$ O(n^2) $, 空间复杂度为$ O(1) $, 不需要额外占用空间

另外希尔排序的最差时间复杂度是$ O(n^2) $, 平均时间复杂度是$ O(nlog{n}) $ 空间复杂度同样为$ O(1) $, 不需要额外占用空间
